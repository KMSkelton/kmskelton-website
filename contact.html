HTML and CSS have been constant companions for over two decades. Books and tutorials comprised the bulk of my "training" until I attended on-site classes at Code Fellows in Seattle. There I learned the fundamentals of JavaScript and completed an 8-week intensive course in Front End Web Development.


While I'm still a junior-level developer, I have a solid training base, know how to ask the questions to figure out a solution to my problem, and know how to use the tools. That's a far cry from my experience with self-teaching.


I offer passion for responsive web design. I build for a quality mobile experience that takes advantage of media queries and relative sizing. The experience improves with the added real-estate of a full size monitor. HTML and CSS comprise the core of my skillset; animations and SASS expand these capabilities. JavaScript (along with jQuery and Angular) add to interactivity.
